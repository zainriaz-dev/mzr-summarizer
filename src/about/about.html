✨ About MZR Summarizer

Free & Open Source

📖 What is this?

AI Page Summarizer is a completely free and open-source Chrome extension that helps you summarize web pages and chat with AI about selected text. It supports multiple AI providers including Google Gemini, OpenAI, and Chrome's built-in AI.

💻 Open Source

This project is open source and available on GitHub. Feel free to contribute, report issues, or fork it!
